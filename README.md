## Catalog - FE

## 1. Install

*a) Install NodeJS [14.16.1]()*

`https://nodejs.org/en/download`

*b) Install Angular CLI*

`npm i -g @angular/cli`

c) Install SASS *(Dart Sass)*

`npm i -g sass`

## 2. Run

a) Build and update css

`npm run-script sass-watch`

b) Run server another tab

`npm start`

## 3. Deployment

a) Server 69

`./restart.sh`
